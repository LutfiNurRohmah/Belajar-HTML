Color Values

Same as color name "Tomato":

rgb(255, 99, 71) RGB Value

#ff6347 HEX Value

hsl(9, 100%, 64%) HSL Value

Same as color name "Tomato", but 50% transparent:

rgba(255, 99, 71, 0.5) RGBA Value

hsla(9, 100%, 64%, 0.5) HSLA Value

In addition to the predefined color names, colors can be specified using RGB, HEX, HSL, or even transparent colors using RGBA or HSLA color values.

HTML RGB and RGBA Colors

RGB Color Values

FORMULA: rgb(red, green, blue), value between 0 and 255

red: rgb(255, 0, 0), green: rgb(0, 255, 0), blue: rgb(0, 0, 255), black: rgb(0, 0, 0), white: rgb(255, 255, 255)

Example:

rgb(238, 130, 238)

rgb(255, 165, 0)

Shades of Grey

Shades of gray are often defined using equal values for all three parameters:

Example:

rgb(60, 60, 60)

rgb(100, 100, 100)

rgb(140, 140, 140)

RGBA Color Values

FORMULA: rgb(red, green, blue, alpha)

The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all)

Example:

rgba(255, 99, 71, 0)

rgba(255, 99, 71, 0.2)

rgba(255, 99, 71, 0.4)

rgba(255, 99, 71, 0.6)

HTML HEX Colors

HEX Color Values

FORMULA: #rrggbb rr:red, gg:green, bb:blue (using a hexadecimal value between 00 and ff)

red: #ff0000, green: #00ff00, blue: #0000ff, black: #000000, white: #ffffff

Example:

#3cb371

#ee82ee

Shades of Grey

Shades of gray are often defined using equal values for all three parameters:

Example:

#686868

#a0a0a0

#bebebe

HTML HSL and HSLA Colors

HSL Color Values

FORMULA: hsl(hue, saturation, lightness)

Hue: degree on the color wheel (0 to 360), (red:0, green:120, blue:240)

Saturation (0%-100%), (0%: shade of gray, 100%: full color)

Lightness (0%-100%), (0%: black, 100%: white)

Example:

hsl(240, 100%, 50%)

hsl(147, 50%, 47%)

hsl(300, 76%, 72%)

Shades of Grey

Shades of gray are often defined by setting the hue and saturation to 0, and adjust the lightness from 0% to 100% to get darker/lighter shades

Example:

hsl(0, 0%, 40%)

hsl(0, 0%, 60%)

hsl(0, 0%, 70%)

HSLA Color Values

FORMULA: hsla(hue, saturation, lightness, alpha)

The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all)

Example:

hsla(9, 100%, 64%, 0)

hsla(9, 100%, 64%, 0.2)

hsla(9, 100%, 64%, 0.4)

hsla(9, 100%, 64%, 0.6)

hsla(9, 100%, 64%, 0.8)
